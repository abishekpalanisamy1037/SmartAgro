# SmartAgro
Run the SmartAgro.html file


The Smart Crop Advisory System is a web application that helps small and marginal farmers choose the best crop based on factors such as soil type, season, rainfall, temperature, and available water. The system also provides fertilizer recommendations, irrigation tips, pest alerts, and estimated crop yield to improve productivity.
Objectives
Recommend suitable crops based on farm conditions.
Suggest fertilizers and irrigation schedules.
Provide pest and disease prevention tips.
Display weather-based farming advice.
Help farmers increase yield and reduce farming risks.
